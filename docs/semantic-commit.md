# Semantic Commit

```md
<type> (<scope>): <description>
```

## Types

* Feat: Features
* Fix: Bug fixes
* Revert: Reverts a previous commit
* Bump: Change version
* CI: Pipelines change
* Refactor: Code change
* Docs: Code documentation
* Style: Refactor code style
* Chore: Description change
* Optimize: Improved code performance
